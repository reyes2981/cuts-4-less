# README

### Cuts4Less

Welcome to Cuts4Less! 

An app created for customers to schedule appointments with the local business, Cuts 4 Less.

### Installation
Clone the repo

`git@github.com:reyes2981/cuts-4-less.git`

Next,

execute `bundle install`

### Usage

`cd cuts-4-less`

`rails s`

Finally navigate to to

http://localhost:3000/
